# 黑门战争历史

## 突袭——日期：2023.8.29
在2023.8.29，忘川下定决心要在一天内打压黑水，事实证明他做到了，黑门由陈词，鹿乃，铃鱼的带领下打死了俩个假黑水，并且曝出了许多黑料证实了刘焰的丑陋嘴脸，并且黑水内部发放讨贼缴文，刘焰名字一落千丈，人人唾弃，最后已黑门，黑水获胜，熊猫跑路收场
